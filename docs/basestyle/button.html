不同用途的按钮

我们给出了7种不同视觉效果的按钮，你可以根据业务场景选择合适的按钮来表达按钮行为的意义。.ui-btn是所有按钮都必须添加的样式，它提供一种默认的视觉效果，其它按钮都是在其基础上另外增加样式，如下表所示：

按钮 样式 描述 html代码
ui-btn 默认的灰色按钮，并带有渐变色
ui-btn ui-btn-primary 提供更深的色度来表明在按钮组中是最重要的行为
ui-btn ui-btn-info 默认样式的一种替代方案
ui-btn ui-btn-success 表示一个成功的或者有积极意义的行为
ui-btn ui-btn-warning 表明警告，并需要采取这种行为的操作
ui-btn ui-btn-danger 表明一个危险的或者潜在的有消极意义的行为
ui-btn ui-btn-inverse 深灰色的按钮，默认按钮的一种替代按钮，跟语义无关的操作
ui-btn ui-btn-link 不强调是个按钮，使它看起来像个链接然但仍然保持按钮的行为

按钮大小

我们提供了四种尺寸的按钮样式，如果想使用大尺寸的按钮，只需要额外添加样式.ui-btn-large，默认尺寸则如上面的表格中展示的那样不需另外添加尺寸相关样式。下面以Primary按钮为例：

如果要创建块级按钮，即单个按钮占据一行，请添加样式.ui-btn-block，那么这个按钮将会填满父级元素的整个空间。

禁用状态

如果要表示此按钮不可用，请添加.ui-disabled样式，如果是<button>元素，则另外添加disabled属性来禁用它。
